# The Vercel AI SDK

---

## What is the AI SDK

**AI SDK** is a free, open-source TypeScript toolkit designed to build AI-powered applications, chatbots, and agents using frameworks like React, Next.js, Vue, and Svelte.

## Key Components & Features:

- **AI SDK Core**: Provides backend functions like `generateText()` and `streamText()` for Node.js, Serverless, and Edge environments.

- **AI SDK UI**: Offers frontend hooks (`useChat`, `useCompletion`, `useObject`) to easily integrate AI into user interfaces.

- **Multi-Provider Support**: Seamlessly switch between providers like OpenAI, Anthropic, and Google Generative AI.

- **Structured Data & Tools**: Supports generating structured JSON/objects and building agentic loops.

- **Integration**: Designed for the Vercel ecosystem, including AI Gateway and RAG workflows.

## To install the Vercel AI SDK

```bash
npm install ai
```
